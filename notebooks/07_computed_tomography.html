Computed Tomography¶

import numpy as np
import scipy.sparse.linalg as la
import matplotlib.pyplot as plt
from scipy.io import loadmat

The following example is a tomographic X-ray data of a walnut. The dataset was prepared by the Finnish Inverse Problems Society.

Measurement Matrix \(A\)¶

data = loadmat('./data/Data82.mat')

A = data['A']

Compute Least Squares Solution¶

b = sinogram.reshape([82*120,1],order='F')

result = la.lsmr(A,b,atol=1e-10,btol=1e-10)

x = result[0]

plt.figure(figsize=(8,8))
plt.imshow(x.reshape(82,82).T,cmap='bone')
plt.show()
